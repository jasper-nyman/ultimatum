# Unity Project Browser Enhancer 

Thank you for using this tool!

I created it to make navigating through Unity a tiny bit easier, giving it
a touch of - in a way - navigating through the Windows Explorer.
Hopefully it will "enhance" your browsing experience as well ;)


Default Hotkeys:
Next Menu Item - CTRL + Shift + E
Previous Menu Item - CTRL + Shift + Q
Next Menu Item (Inspector Locked) - Shift + ALT + E
Previous Menu Item (Inspector Locked) - Shift + ALT + Q
Toggle Project Browser Helper Arrows - CTRL + Shift + "-"

I recommend rebinding the default hotkeys.

To rebind shortcuts (You can even bind them to your mouse buttons), go to Edit/Shortcuts -> Main Menu or just type Project Browser Enhancer in the search box.

For general use, navigate to Window/Project Browser Pro.
